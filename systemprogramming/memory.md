보호되지만. 메모리 영역에 있는 데이터를 다른 프로세스도 사용할 수 있도록 해야할 경우도 필요.

위 설비와 같이 pipe등을 이용해서. 데이터 통신을 이용하여 데이터를 전달하는 방법도 있겠지만,
